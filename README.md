# Cameroon Migration Analysis (2015–2024)

## Project goal
Analyze the evolution of Cameroonian international migration between 2015 and 2024, with a focus on destinations, reasons for entry, and post-arrival trajectories.

## Main questions
Q1. How did the main destination countries evolve?
Q2. How did observable reasons for entry evolve?
Q3. How did post-arrival trajectories evolve?

## Data sources
- UN DESA
- Eurostat
- Statistics Canada / IRCC
- DHS / OHSS
- Japan immigration / visa statistics

- data/raw
- data/processed
- notebooks
- src
- outputs
- docs
